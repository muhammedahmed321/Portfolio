Project Overview

The Ice Cream Parlour Management System is a full-stack web application developed using ASP.NET Core MVC and C#. The system provides a complete digital platform for customers to explore ice cream flavors, place orders, subscribe to plans, and submit recipes. It also includes a powerful admin panel to manage users, payments, products, feedback, and website content efficiently. The project focuses on enhancing user experience while automating business operations such as order management, payment tracking, and customer engagement.

Key Features

  • 🔐 User Authentication System Registration, Login, Logout functionality
  • 🛒 E-commerce Functionality Add to Cart Order Placement Payment Integration
  • 📦 Subscription Plans Monthly & yearly subscription system
  • 🍦 Flavor & Recipe Management Users can submit recipes Admin can manage flavors & categories
  • 🧑‍💼 Admin Panel User Management Product & Category Management Payment Tracking Feedback & Contact Messages Handling
  • ⭐ Feedback System User ratings and reviews
  • 📩 Contact & FAQ Management Dynamic FAQ section Contact form handling
  • 📚 Recipe Submission System Users can submit their own recipes
